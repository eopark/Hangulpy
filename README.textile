h1. 한글파이

h2. 개요

한글파이는 한글 유니코드 기능을 제공하는 파이썬 모듈입니다.

h3. 자동 조사/접미사 첨부

정해진 규격을 지키면 주격, 도구격, 서술격 등 모든 조사를 앞 단어에 알맞게 자동으로 바꿔 줍니다.

h3. 자모 분해 및 결합

겹받침, 반자음 + 모음, 'ㅣ' 역행동화가 일어난 모음 등을 자동화에 필요한 상황에 맞게 쓸 수 있도록 분해하고 결합합니다.

h3. 두음법칙을 적용한 한자 음 읽기 (미구현)

단어 속 위치에 따라 다르게 소리나는 한자를 자동으로 잡아 줍니다. (미구현)
